# GTranslate

# Installation

You can install using this package either npm or yarn

```
npm install https://github.com/oozou/gtranslate --save-dev
```

or

```
yarn add https://github.com/oozou/gtranslate --dev
```

# Usage

```
./node-modules/.bin/gtranslate
```
or

```
yarn gtranslate
```
